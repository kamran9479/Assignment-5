1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. We imagine it as a tree object and we can interact to change the content, structure, and style of a page. Also we can set various type function and do dynamic thing with it.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that involves handling events at a higher level in the DOM rather than attaching event listeners to individual elements. This is done by leveraging the concept of event bubbling (or event propagation), where an event triggered on a child element "bubbles up" to its parent elements, allowing a parent element to listen for events on its child elements.
It is useful beucouse Instead of adding an event listener to each child element (which could be numerous and consume more memory and processing), you only need one listener on the parent. This is especially useful when dealing with a large number of elements, like a long list or dynamically generated content.
